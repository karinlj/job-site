# hyrule-jobs

## Description

Application for exploring Vue.js 3 with Typescript.

![Screenshot](/src/assets/screenshot.jpg?raw=true "Screenshot")

## Prerequisites

NodeJS

## Installation

To clone

`git clone https://github.com/karinlj/job-site.git`

```
npm install
```

## To run

```
npm run serve
```

## To build

```
npm run build
```

Then use the dist folder you get.

## Built With

- Vue.js 3
- Typescript
- CSS
